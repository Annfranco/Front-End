HTML

HTML é a sigla para HyperText Markup Language, que significa "Linguagem de Marcação de Hipertexto". É uma linguagem de computador que define a estrutura e o significado do conteúdo de uma página web.


Além do HTML, outras tecnologias são usadas para descrever a aparência e a funcionalidade de uma página da web. Por exemplo, o CSS é usado para descrever a aparência e o JavaScript para descrever a funcionalidade.
