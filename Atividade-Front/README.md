# Atividade

## Instruções:
1. Crie 3 telas:
    - Bem-vindo
    - Cadastro
    - Pacientes cadastrados

2. Crie 3 botões:
    - Nome
    - Telefone
    - CPF
    - CEP
    - E-mail
    - Senha

3. A aplicação deve ter uma tela para mostrar os usuários cadastrados
    - Sem exibir a senha do paciente.
